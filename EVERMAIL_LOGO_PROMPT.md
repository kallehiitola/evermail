# Evermail Logo Design Prompt

## Brand Name
**Evermail** (one word, lowercase 'e')

## Company Overview
Evermail is a modern SaaS platform that enables users to upload, view, search, and analyze email archives from .mbox files. It's a cloud-based solution targeting individuals, small businesses, and enterprises who need powerful email archiving with AI-powered search capabilities.

## Design Process Recommendations

### Step 1: Concepts (3-5 variations)
Create initial sketches/mockups of 3-5 different logo concepts exploring:
- Different visual metaphors (infinity, search, archive, cloud, letter E)
- Different color palettes (blue, purple, teal)
- Different styles (geometric, abstract, typographic)

### Step 2: Refinement (2 finalists)
- Select 2 strongest concepts
- Refine details, test at multiple sizes
- Test on light/dark backgrounds
- Get feedback from target audience

### Step 3: Final Selection
- Choose winning concept
- Create all variations (full, icon, wordmark, monochrome)
- Test in real-world contexts (browser tab, app icon, website header)

### Step 4: Brand Guidelines
- Document color codes, typography, spacing
- Create usage examples (correct/incorrect)
- Provide all file formats
